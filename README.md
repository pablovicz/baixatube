## Como utilizar:

<p>1 - Clone o repositório atual</p>

```bash
$ git clone "https://github.com/pablovicz/baixatube.git"
```

<p>2 - No diretório /baixatube-service, execute:</p>


```bash
$ venv\Scripts\activate.bat 
$ python app.py
```

<p>3 - No diretório /baixatube-webapp, execute:</p>

```bash
$ npm install 
$ npm start 
```
